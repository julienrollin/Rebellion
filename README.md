# Rebellion

Production UI extensions for Guerilla Render.

## Features

- **Gizmo 2.0**: cleaner move, rotate, scale handles.
- **Color Picker 2.0**: wheel, sliders, temperature, raw/view RGB toggle.
- **Auto Save**: background scene backups, skips unchanged scenes.
- **Easy TriPlanar**: one-texture triplanar node with random tiling and color correction.

## Install

Double-click:

```text
Install_Rebellion_CreativeSeeds.bat
```

Target:

```text
C:\CreativeSeeds\Applications\guerilla2\app
```

Dry run:

```text
.\Install_Rebellion_CreativeSeeds.bat /dryrun
```

Installer backs up replaced files. Restart Guerilla after install.
